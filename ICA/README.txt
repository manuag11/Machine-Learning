To run the project, navigate to the folder src and run the file blind_source_separation.m. Alternatively, you can type the following on the MATLAB terminal:
blind_source_separation

blind_source_separation.m - It is the file which consolidates the various segments of the project
ica.m - This file performs Independent Component Analysis
rescale.m - This file rescales the signals
plot_signals.m - This file plots the original, mixed and recovered signals.
mix_signals.m - This file mixes the source signals.
